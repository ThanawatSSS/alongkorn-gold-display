# Gold Price Display - ร้านทองอลงกรณ์

เว็บแอปพลิเคชันสำหรับแสดงราคาทองคำแบบ Real-time ออกแบบมาเป็นพิเศษเพื่อใช้งานบน Smart TV รุ่นเก่า (Samsung TV, Tizen, ฯลฯ) ที่ร้านทองอลงกรณ์

## คุณสมบัติ (Features)

- แสดงราคาทองคำจากสมาคมค้าทองคำ (รับซื้อ/ขายออก)
- คำนวณราคาขายหน้าร้านโดยอัตโนมัติ
- แสดงสถานะราคา ขึ้น/ลง พร้อมสีสันที่ชัดเจน
- บอกระยะเวลาหลังจากที่มีการอัปเดตราคาล่าสุด (เช่น อัปเดตเมื่อ 5 นาทีที่แล้ว)
- รองรับเบราว์เซอร์รุ่นเก่า (ES5 Stealth Mode)
- รองรับการแสดงผลบน GitHub Pages
